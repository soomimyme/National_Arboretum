### section 2

**국립수목원**

- 국립수목원은 1920년대부터 시작된 우리나라 산림생물종 연구의 전통을 잇고 있으며 특히 우리나라 산림생물 주권을 확보하는데 중요한 역할을 수행하고 있습니다. 또한 조선 시대 세조대왕 능림으로 지정된 1468년 이래로 550여 년 이상 자연 그대로 보전되어 오고 있는 광릉숲을 보호·관리하고 있는 산림청 소속의 국립연구기관입니다.
